gRPC Kotlin Cloud Run Example
-----------------------------

Run Locally:
1. In one shell / terminal window, start the server:
    ```
    ./gradlew run
    ```
1. In another shell / terminal window, run the client:
    ```
    ./gradlew HelloWorldClient
    ```
